### 2. (WHAT)都有什么功能

1. 主动发送信息

    * 以RESTful API的形式提供发信息接口
    * 解决了获取token和集成微信接口的问题
    * 传参时可指定APP的参数来切换不同的通知载体

2. 被动接收指令

    * 捕获用户在APP上的操作行为
    * 点击事件、点击跳转网页事件和回复文本信息处理
